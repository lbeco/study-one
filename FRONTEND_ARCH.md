# study-one 前端技术架构

## 技术选型

### 核心框架
- **Vue 3** - 前端框架
- **Vue Router** - 路由管理
- **Pinia** - 状态管理
- **Vite** - 构建工具

### UI 组件库
- **Element Plus** - 功能丰富，文档完善，支持中文

### 第三方库
- **Axios** - HTTP 客户端
- **Monaco Editor** - 代码编辑器（用于 Markdown 编辑）
- **Marked** - Markdown 解析
- **VueUse** - 实用工具集

## 项目结构

```
frontend/
├── public/                # 静态资源
├── src/
│   ├── assets/            # 资源文件
│   │   ├── styles/        # 全局样式
│   │   └── icons/         # 图标
│   │
│   ├── components/        # 通用组件
│   │   ├── common/        # 基础组件
│   │   └── layout/        # 布局组件
│   │
│   ├── views/             # 页面组件
│   │   ├── dashboard/     # 仪表盘
│   │   ├── folders/       # 文件夹管理
│   │   ├── knowledge/     # 知识项管理
│   │   ├── markdown/      # Markdown 编辑器
│   │   ├── webpage/        # 网页内容管理
│   │   └── tags/           # 标签管理
│   │
│   ├── router/            # 路由配置
│   │   └── index.js
│   │
│   ├── stores/            # Pinia 状态管理
│   │   ├── folder.js       # 文件夹状态
│   │   ├── knowledge.js    # 知识项状态
│   │   ├── tag.js          # 标签状态
│   │   └── user.js         # 用户状态
│   │
│   ├── services/           # API 服务
│   │   ├── api.js          # API 配置
│   │   ├── folder.js       # 文件夹 API
│   │   ├── knowledge.js    # 知识项 API
│   │   └── tag.js          # 标签 API
│   │
│   ├── utils/              # 工具函数
│   │   ├── request.js      # 请求封装
│   │   └── helpers.js      # 辅助函数
│   │
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
│
├── .env                    # 环境变量
├── vite.config.js          # Vite 配置
└── package.json            # 依赖管理
```

## 页面设计

### 1. 整体布局

#### 顶部导航栏
- Logo 和应用名称
- 搜索框
- 用户信息/设置

#### 侧边栏
- 文件夹树结构
- 快捷操作（新建文件夹、新建知识项）
- 标签列表

#### 主内容区
- 根据当前路由显示对应页面

### 2. 仪表盘页面
- 最近创建/修改的知识项
- 知识项分类统计
- 学习进度概览

### 3. 文件夹管理页面
- 文件夹树结构显示
- 文件夹的创建、编辑、删除
- 拖拽排序功能

### 4. 知识项列表页面
- 知识项卡片列表
- 支持按文件夹、类型、标签筛选
- 支持排序（创建时间、修改时间）
- 批量操作（删除、移动）

### 5. Markdown 编辑器页面
- Monaco 编辑器（支持语法高亮）
- 实时预览
- 保存功能
- 标签管理
- 文件夹选择

### 6. 网页内容管理页面
- 网页信息显示
- URL 输入
- 网页抓取功能
- 内容编辑

### 7. 标签管理页面
- 标签列表
- 标签的创建、编辑、删除
- 标签颜色设置

## API 集成

### 基础 API 调用
- 使用 Axios 封装 API 调用
- 统一的错误处理
- 加载状态管理

### 具体 API 集成
1. **文件夹 API**
   - 获取文件夹树
   - 增删改文件夹

2. **知识项 API**
   - 获取知识项列表
   - 搜索知识项
   - 增删改知识项

3. **Markdown API**
   - 获取 Markdown 内容
   - 更新 Markdown 内容
   - 上传 Markdown 文件

4. **网页 API**
   - 获取网页内容
   - 更新网页内容
   - 抓取网页

5. **标签 API**
   - 获取标签列表
   - 增删改标签

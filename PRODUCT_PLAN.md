# study-one 产品方案

## 项目概述
study-one 是一个能够自主发现可学习新知识的本地知识管理平台。

## 核心功能

### 1. 知识管理模块
- **知识类型支持**
  - Markdown 文件
  - 网页链接（自动抓取/保存）

- **知识组织**
  - 文件夹/目录结构
  - 标签系统
  - 搜索功能

### 2. 知识整理与更新模块
- **自动关联**
  - 基于现有知识内容，智能检索相关网页资源
  - 提取网页关键信息

- **知识总结**
  - 自动生成知识摘要
  - 对比新旧知识，提供更新建议

### 3. 主动学习与测试模块
- **智能提问**
  - 基于知识库内容生成问题
  - 识别知识盲区
  - 自适应难度调整

- **学习进度追踪**
  - 记录学习状态
  - 提供学习建议

## 技术选型与产品决策

### 技术栈
- **前端框架**：Vue
- **后端语言**：Python
- **本地数据库**：SQLite
- **数据存储**：混合模式（本地为主，可选云端同步）

### 功能决策
- **网页抓取**：只保存关键信息+链接（节省空间）
- **知识更新**：人工确认（提供更新建议，确认后才更新）
- **提问形式**：混合（选择题+简答题等多种形式）
- **知识盲区识别**：两者结合（答题情况+知识关联分析）

### 产品形态
- **部署方式**：本地 Web 应用
- **用户模式**：单用户
- **优先级**：知识管理模块优先实现
- **使用场景**：个人学习+工作资料都有
